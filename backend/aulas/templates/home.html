Bienvenidos a mi app Aulas 2024

Este proyecto está diseñado para agilizar y gestionar la programación académica y las reservas de aulas de manera eficiente. Desarrollado con Django, presenta un sistema API seguro protegido por autenticación JWT, lo que permite a los administradores y al personal manejar cursos, profesores, aulas y reservas sin problemas. Nuestro objetivo es proporcionar una plataforma intuitiva que mejore la experiencia educativa al simplificar las tareas administrativas. Explore la documentación de la API y el panel de administración para obtener más información sobre las funcionalidades y capacidades de este sistema.
